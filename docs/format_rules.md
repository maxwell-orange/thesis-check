# 格式规则说明

## 山东财经大学本科毕业论文格式规范

### 1. 页面设置

| 项目 | 要求 |
|------|------|
| 纸张大小 | A4 |
| 纸张方向 | 纵向 |
| 上边距 | 2.5 cm |
| 下边距 | 2.0 cm |
| 左边距 | 2.5 cm |
| 右边距 | 2.0 cm |

### 2. 章标题格式

| 项目 | 要求 |
|------|------|
| 字体 | 黑体 |
| 字号 | 三号 (16pt) |
| 加粗 | 是 |
| 对齐 | 居中 |
| 段前间距 | 0.5 行 |
| 段后间距 | 0.5 行 |

**格式示例**:
```
第一章 绪论
```

### 3. 节标题格式

| 项目 | 要求 |
|------|------|
| 字体 | 黑体 |
| 字号 | 四号 (14pt) |
| 加粗 | 是 |
| 对齐 | 左对齐 |
| 段前间距 | 0.5 行 |
| 段后间距 | 0.5 行 |

**格式示例**:
```
1.1 研究背景
```

### 4. 小节标题格式

| 项目 | 要求 |
|------|------|
| 字体 | 黑体 |
| 字号 | 小四 (12pt) |
| 加粗 | 是 |
| 首行缩进 | 2 字符 |
| 段前间距 | 0.5 行 |
| 段后间距 | 0.5 行 |

**格式示例**:
```
  1.1.1 国内外研究现状
```

### 5. 正文格式

| 项目 | 要求 |
|------|------|
| 中文字体 | 宋体 |
| 英文字体 | Times New Roman |
| 字号 | 小四 (12pt) |
| 行距 | 1.5 倍 |
| 首行缩进 | 2 字符 |
| 对齐 | 两端对齐 |

### 6. 摘要格式

**中文摘要**:
- 字数：300 字左右（250-350 字）
- 字体：宋体
- 字号：小四

**英文摘要**:
- 应与中文摘要内容对应
- 字体：Times New Roman
- 字号：小四

### 7. 参考文献格式

**数量要求**:
- 最少 10 篇
- 近三年文献比例 ≥ 30%

**格式标准**: GB/T 7714

**文献类型标识**:

| 标识 | 文献类型 |
|------|----------|
| [J] | 期刊 |
| [M] | 专著 |
| [D] | 学位论文 |
| [C] | 会议论文 |
| [N] | 报纸 |
| [R] | 报告 |
| [S] | 标准 |
| [EB/OL] | 网络文献 |

**示例**:
```
[1] 张三, 李四. 论文标题[J]. 期刊名称, 2023, 45(3): 123-130.
[2] 王五. 书名[M]. 北京: 出版社, 2022.
```

### 8. 图表格式

**图标题**:
- 位置：图下方
- 字体：宋体
- 字号：五号 (10.5pt)
- 加粗：是
- 编号：按章编号，如"图1-1"

**表标题**:
- 位置：表上方
- 字体：宋体
- 字号：五号 (10.5pt)
- 加粗：是
- 编号：按章编号，如"表2-1"

### 9. 目录格式

**目录标题**:
- 字体：黑体
- 字号：小二号
- 对齐：居中

**目录内容**:
- 章标题：黑体，小四
- 其他：宋体，小四

## 已知限制

1. **python-docx 限制**:
   - 无法读取页眉页脚内容
   - 页码检查可能不准确
   - 复杂表格格式识别能力有限

2. **字体识别**:
   - 部分文档可能无法准确识别字体名称
   - 需要人工复核
